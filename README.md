# DevOps Roadmap (Arabic)

هذا المشروع هو **خارطة طريق عملية لتعلّم DevOps** موجهة للمجتمع العربي،  
لكن بأسلوب مختلف عن الشروحات التقليدية.

هنا ما نركز على:

❌ الحفظ 

❌ الأوامر بدون فهم  

نركز على: 


✅ الفهم 

✅ التجربة 

✅ كسر الأشياء ثم إصلاحها  

---

##  هدف المشروع
مساعدة أي شخص مهتم بـ DevOps إنه:
- يفهم ليش المشاكل تصير
- يتعامل مع الفشل بثقة
- يكتسب عقلية DevOps الحقيقية

هذا المشروع **تعليمي + عملي**،  
ومبني على سيناريوهات تشبه بيئة العمل الواقعية.

---

##  محتوى المشروع (قيد البناء)
> سيتم تحديث المحتوى بشكل تدريجي

- Linux Basics
- Docker & Containers
- CI/CD Pipelines
- Kubernetes
- Infrastructure as Code (Terraform)
- Monitoring & Logging

---

##  أسلوب التعلّم
كل قسم يحتوي على:
- شرح مبسط
- أخطاء شائعة
- سيناريوهات فشل
- كيف تصلح المشكلة
- ليه صارت من الأساس

---

##  لمن هذا المشروع؟
- طلاب DevOps
- مبتدئين في المجال
- Developers حابين يفهمون الـ Ops
- أي شخص يبي يتعلم صح

---

## المساهمة
المشروع مفتوح للمجتمع:
- اقتراحات
- Issues
- Pull Requests

الهدف إننا نبني **مرجع عربي قوي** مع بعض.

---

##  الحالة
 المشروع في بدايته  
والمحتوى قادم خطوة بخطوة.
