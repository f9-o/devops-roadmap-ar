# DevOps Roadmap (Arabic)

خارطة طريق عربية تساعدك تتعلّم **DevOps خطوة بخطوة**  
من الأساسيات إلى مستوى جاهز لسوق العمل.

هذه ليست دورة،  
بل مسار تعلّم عملي يوضّح:
- ماذا تتعلّم؟
- بأي ترتيب؟
- ولماذا هذا مهم؟

---

##  هدف المشروع
مساعدة أي شخص مهتم بـ DevOps على:
- فهم المسار الصحيح للتعلّم
- تجنّب التشتت
- بناء أساس قوي في DevOps
- الانتقال من Developer إلى DevOps-ready

---

##  كيف تستخدم هذه الرودماب؟
1. اتبع الأقسام بالترتيب.
2. لا تنتقل لقسم قبل فهم اللي قبله.
3. جرّب بيدك (حتى لو أخطأت).
4. ارجع للرودماب كل ما ضعت.

---

##  خريطة الطريق

### 1️⃣ Linux Fundamentals
**لماذا؟**  
لأن أغلب بيئات DevOps مبنية على Linux.

**تتعلّم:**
- أوامر أساسية
- إدارة الملفات والصلاحيات
- العمليات والخدمات
- أخطاء شائعة في السيرفرات

---

### 2️⃣ Networking Basics
**لماذا؟**  
DevOps بدون Networking = عمى.

**تتعلّم:**
- TCP/IP
- DNS
- Ports
- Debugging مشاكل الاتصال

---

### 3️⃣ Docker & Containers
**لماذا؟**  
لأن الحاويات أساس الـ CI/CD و Kubernetes.

**تتعلّم:**
- Docker fundamentals
- Images & Containers
- أخطاء التشغيل
- مشاكل الشبكة داخل الحاويات

---

### 4️⃣ CI / CD Pipelines
**لماذا؟**  
لأن الأتمتة هي قلب DevOps.

**تتعلّم:**
- CI vs CD
- بناء pipelines
- Debugging فشل الـ builds
- أخطاء النشر الشائعة

---

### 5️⃣ Kubernetes
**لماذا؟**  
لأن أغلب الأنظمة الحديثة تعمل عليه.

**تتعلّم:**
- مفاهيم K8s
- Deployments & Services
- مشاكل Pods
- Debugging الإنتاج

---

### 6️⃣ Infrastructure as Code (Terraform)
**لماذا؟**  
لأن البنية التحتية لازم تكون قابلة للتكرار.

**تتعلّم:**
- مفاهيم IaC
- Terraform basics
- State management
- أخطاء مدمّرة شائعة

---

### 7️⃣ Monitoring & Logging
**لماذا؟**  
لأنك ما تقدر تصلّح شيء ما تشوفه.

**تتعلّم:**
- Monitoring concepts
- Logging
- Alerting
- قراءة المشاكل قبل وقوعها

---

##  الجانب العملي
كل قسم يحتوي على:
- شرح مبسّط
- أمثلة
- أخطاء شائعة
- سيناريوهات فشل واقعية
- كيف تصلّح المشكلة

---

##  لمن هذه الرودماب؟
- مبتدئين في DevOps
- Developers يبغون ينتقلون لـ DevOps
- أي شخص ضايع وما يعرف من وين يبدأ

---

##  المساهمة
الرودماب مفتوحة للمجتمع:
- اقتراح تحسينات
- إضافة موارد
- Pull Requests

---

##  الحالة الحالية
 الرودماب قيد البناء  
وسيتم تطويرها تدريجيًا مع المجتمع.
